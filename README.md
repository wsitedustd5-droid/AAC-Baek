# abcd# 🗣️ 올인원 AAC - 소통 도우미

실어증, 뇌병변 장애, 자폐 등 의사소통에 어려움을 겪는 분들을 위한 **보완대체 의사소통(AAC)** 웹 애플리케이션입니다.

![Version](https://img.shields.io/badge/version-2.0.0-orange)
![License](https://img.shields.io/badge/license-MIT-blue)

---

## ✨ 주요 기능

### 💬 말하기 (핵심 기능)
- **카테고리별 카드**: 행동, 감정, 요청, 통증, 장소, 사람, 음식, 시간
- **긴급 버튼**: 도움, 아픔, 화장실 등 즉시 전달이 필요한 메시지
- **통증 스케일**: 10단계 통증 정도 표현
- **시제/존댓말 변환**: 카드 롱프레스로 시제 변경 (과거/현재/미래) 및 높임말/반말 선택
- **커스텀 카드**: 사용자가 직접 카드 추가 가능
- **AI 단어 추천**: 선택한 단어 다음에 올 말 자동 추천 (준비 중)

### 🎤 음성 인식 (스마트 기능)
- **상대방 말 듣기**: 마이크 버튼으로 상대방 말 인식
- **AI 답변 추천**: 들은 내용에 대한 적절한 답변 추천 (준비 중)

### 📱 사용자 경험
- **청자 모드**: 큰 화면으로 메시지 전달
- **음성 출력**: TTS로 메시지 음성 재생
- **사용 기록**: 자주 쓰는 문장 빠른 재사용
- **검색 기능**: 단어/문장 빠른 검색
- **다크모드**: 눈부심 방지 고대비 모드
- **글자 크기 조절**: 작게/보통/크게

## 🎨 디자인 개선 사항 (v2.0)

### 현대적인 인터페이스
- ✅ **부드러운 애니메이션**: 카드 선택 시 바운스 효과, 페이드인 등
- ✅ **글래스모피즘**: 헤더 블러 효과로 모던한 느낌
- ✅ **그라데이션 버튼**: Primary 버튼에 그라데이션 적용
- ✅ **마이크로 인터랙션**: 호버/클릭 시 즉각적인 피드백

### 접근성 최적화
- ✅ **큰 터치 영역**: 최소 44x44px 터치 영역
- ✅ **명확한 시각 피드백**: 선택 시 애니메이션과 색상 변화
- ✅ **고대비 색상**: WCAG 기준 준수
- ✅ **즉각적인 반응**: 0.2초 이하 애니메이션으로 빠른 피드백

### 반응형 디자인
- ✅ **모바일 최적화**: 360px ~ 500px 최적화
- ✅ **태블릿 지원**: 768px+ 4열 그리드
- ✅ **터치 제스처**: 스와이프로 페이지 전환
